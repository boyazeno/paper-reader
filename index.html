Paper Reader — install via apt

Debian/Ubuntu (amd64):

sudo install -d -m 0755 /etc/apt/keyrings
curl -fsSL https://boyazeno.github.io/paper-reader/paper-reader.gpg | sudo tee /etc/apt/keyrings/paper-reader.gpg >/dev/null
echo "deb [arch=amd64 signed-by=/etc/apt/keyrings/paper-reader.gpg] https://boyazeno.github.io/paper-reader stable main" \
  | sudo tee /etc/apt/sources.list.d/paper-reader.list
sudo apt update
sudo apt install paper-reader

Or download the .deb/.AppImage directly from the Releases page.
